
# Single Page HTML Resume

A clean, structured, and semantic HTML-only resume template. This project demonstrates how to build a readable and organized layout using pure HTML tags without any CSS.

## Features
- **Semantic HTML**: Uses proper tags like `<header>`, `<section>`, `<table>`, `<ul>`, etc.
- **No CSS**: Layout relies solely on browser defaults and HTML attributes.
- **Responsive-friendly**: Flows naturally on different screen sizes (default browser behavior).

## File Structure
- `resume.html`: The main resume file.

## Usage
1. Download or clone this repository.
2. Open `resume.html` in any web browser (Chrome, Firefox, Edge, Safari).
3. The resume will render with a classic, structured document style.

## Customization
To use this for your own resume, simply edit the text content within the HTML tags in `resume.html`.
